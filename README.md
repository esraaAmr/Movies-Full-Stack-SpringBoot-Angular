# Movie Management System  

A full-stack application for managing movies with user authentication, role-based dashboards, and ratings.  

It consists of:  
- **Backend**: Spring Boot REST API (Java 17, PostgreSQL, Liquibase, Swagger)  
- **Frontend**: Angular application (TypeScript 5, responsive UI)

- Add, search, and manage movies

- Responsive Angular dashboard (Admin & User views)  
- Confirmation dialogs for delete operations  
- Swagger UI for interactive API docs
